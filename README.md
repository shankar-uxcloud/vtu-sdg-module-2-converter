# VTU SDG Module-2 Smart Answer Converter 🚀

A fully offline web-based application designed to help VTU students prepare exam-ready answers efficiently.

## 🌐 Live Demo
👉 https://shankar-uxcloud.github.io/vtu-sdg-module-2-converter/

---

## 💡 Features

- 📚 Browse all Module-2 questions with structured answers  
- 🔍 Smart search functionality to quickly find answers  
- ⚡ Compress long notes into 1-page exam format  
- 🌐 Works completely offline (no API required)  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## 🎯 Purpose

This project is built to:
- Help students write **VTU exam-ready answers**
- Save time during preparation  
- Provide structured and easy-to-understand content
